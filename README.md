# 持股金庫 · Stock Vault

自己用的看價工具。**資料留本機、入口輕量密碼、純前端報價、零相依單檔。**

引擎搬自系統 #99 個人持股儀表板；去掉 Python server，改成一隻可獨立部署的 HTML repo。

## 拿到網址就能用

開 live 版 → **https://zonal0223tw-max.github.io/stock-vault/**

1. 第一次進來：**自己設一組密碼**（記住它，之後用來解鎖；忘了可在解鎖畫面「忘記密碼？」重設，資料不會丟）。
2. 初始資料選「範例試玩」先逛逛，或「空白開始」直接填自己的。
3. 點「✏️ 編輯持股」填股號 → 🔄 自動帶名稱／配息月／參考價／月線 → 補張數、成本 → 儲存。
4. 「更新今日報價」抓即時價、算今日損益。

> **你的資料只存在你自己的瀏覽器。** 別人開同一個網址是各自獨立的金庫——你看不到他的、他看不到你的。

## 開啟（自己 clone / host）

報價走公開 CORS proxy，需要有效的網頁 origin——**別直接雙擊 `file://`**（origin 是 null、報價會被擋）。用本機 server 或 host：

```bash
cd stock-vault
python3 -m http.server 8080
# 瀏覽器開 http://localhost:8080
```

或 host 到 GitHub Pages（資料仍只在你的瀏覽器，不進 repo）。

## 第一次

設一組密碼 → 選「範例試玩 / 空白開始 / 匯入 JSON」。範例是程序生成的假持股，點「✏️ 編輯持股」換成你自己的。

## 密碼（輕量，非金庫加密）

- SHA-256 hash 存本機、**全程不上網驗證**。
- 只擋「別人拿你裝置隨手打開」——資料在 localStorage 是明文，不是加密金庫。
- 忘記可在解鎖畫面點「忘記密碼？」重設，**持股資料不會丟**。

## 資料

- 只存在這台裝置的瀏覽器 `localStorage`。
- 「⬇ 匯出」隨時下載明文 JSON 備份（換機、防清快取丟資料）。
- 「⬆ 匯入」還原或跨機搬移。
- **真實持股數字永遠不進這個 repo**（repo 只有引擎 + 程序生成的範例）。

## 報價

- 「更新今日報價」走 `corsproxy.io` → Yahoo Finance。
- **只送股號出門**——張數、成本、總市值永遠不離開本機。
- 編輯持股時點 🔄 自動抓名稱 / 配息月 / 參考價 / 5 年月線（五線譜用）。

## 功能

- 持股總覽：市值 / 成本 / 資本利得 / 總報酬（YTD 含息）/ 今日漲跌
- 五線譜走勢：線性迴歸 ±1σ ±2σ、z 區位（價值～泡沫）、接即時報價末端
- 月份戰情表：已入帳 / 待入帳 / 預估 / 錯過
- 股息明細時間軸、未來 60 天除息預告
- 信託鎖定部位分流（不計入「可動」）
- 含息 / 不含息切換、1 / 3 / 5 年範圍

> 新增一檔持股，上面 KPI 總金額、五線譜、月份戰情、股息明細**全部自動重算**——資料變、全鏈路重算，不是局部補丁。
